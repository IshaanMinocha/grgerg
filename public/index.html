Cloud DevOps Todos API

Welcome to the Todos API documentation. Use the endpoints below to manage your tasks.

GET /health

Check if the API is running correctly.

GET /todos

Retrieve all todos.

GET /todos/:id

Retrieve a specific todo by ID.

POST /todos

Create a new todo.

{
    "title": "Learn Docker",
    "completed": false
}
PUT /todos/:id

Update an existing todo.

{
    "title": "Learn Kubernetes",
    "completed": true
}
DELETE /todos/:id

Delete a todo by ID.
